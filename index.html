1. Difference between HTML tags and elements?

HTML Tags HTML Elements
Tags are the starting and ending part of an HTML element.They begin with < symbol and end with > symbol. Elements enclose the contents in between the tags. They consist of some kind of structure and expression.
Tags are used to hold the HTML element. Elements holds the content.
Ex:bold tag Ex:Content inside bold tag
